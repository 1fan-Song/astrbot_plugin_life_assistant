# 生活助手 (Life Assistant)

综合生活管理插件，帮你记账、写日记、管笔记、追基金、管日程。

## 推荐模型

本插件大量使用 LLM 工具调用（Function Calling），**需要模型支持工具调用能力**。推荐使用：

- **GLM-4/GLM-5 系列**（智谱）— 国内首选，延迟低
- **GPT-4o / GPT-4.1**（OpenAI）
- **Claude 3.5 / Claude 4**（Anthropic）
- **DeepSeek V3**（深度求索）

> 不支持工具调用的模型（如部分小模型、纯对话模型）无法使用记账、笔记、日程等功能。

## 功能模块

### 智能记账
- 自然语言记账："午饭 25"、"咖啡 18"
- 自动分类（餐饮、交通、购物等）
- 月度/周度财务总结 + 图表

### 日记自动生成
- 每晚自动总结当天对话，生成日记
- 自动保存到 Obsidian（可选）
- 支持图片插入（自动保存生活照片，排除游戏截图）

### 笔记系统
- 支持读书、游戏、影视、音乐、追剧笔记
- 自动润色 + 保留原文
- 笔记搜索（按关键词检索已有笔记）
- 自动同步到 Obsidian

### 媒体库
- 管理想读/想玩/想看列表
- 标记完成状态 + 评分
- 自动从列表中移除已完成项目

### 健康追踪
- 记录体重、睡眠、步数、运动等
- 健康趋势分析 + 图表
- 支持健康快照批量录入

### 基金监控
- 实时基金净值查询
- 每日晨报含基金涨跌
- 跌幅预警（可配置阈值）

### 日程管理
- 一次性/周期性日程
- 自然语言时间解析（"明天下午3点"、"每天早上9点"）
- 精确 cron 提醒（非轮询）
- 与晨报/日记联动

### 晨报 & 总结
- 每日晨报：天气、新闻、基金、今日日程
- 晚间总结：当日对话回顾、健康数据、财务摘要
- 周报：一周汇总 + 图表

### Obsidian 同步
- 日记、笔记、财务总结自动写入 Obsidian
- 图片自动保存到附件目录

## 命令

| 命令 | 说明 |
|------|------|
| `/wa <金额>` | 记录资产快照 |
| `/ws [today/week/month/year]` | 财务总结 |
| `/note <类型> <标题>\|<内容>` | 记笔记 |
| `/briefing` | 手动触发晨报 |
| `/summary` | 手动触发每日总结 |
| `/week` | 周报 |
| `/health [体重\|睡眠\|...] <值>` | 记录健康数据 |
| `/fund` | 基金持仓查询 |

## 配置项

| 配置 | 说明 | 默认值 |
|------|------|--------|
| `obsidian_vault_path` | Obsidian 保管库根路径 | 空（不写入） |
| `obsidian_diary_folder` | 日记子文件夹名 | `日记` |
| `obsidian_notes_folder` | 笔记子文件夹名 | `笔记` |
| `obsidian_finance_folder` | 财务子文件夹名 | `财务` |
| `auto_polish` | 自动润色笔记/日记 | `true` |
| `auto_categorize` | 自动分类记账 | `true` |
| `fund_holdings` | 基金持仓（每行一条） | 空 |
| `briefing_push_target` | 晨报推送目标会话 | 空 |
| `briefing_push_hour` | 晨报推送时间（小时） | `8` |
| `summary_push_hour` | 晚间总结推送时间（小时） | `22` |
| `amap_weather_key` | 高德天气 API Key | 空 |
| `amap_weather_city` | 高德天气城市编码 | `440300`（深圳） |
| `weather_city_name` | 天气城市名称（英文） | `Shenzhen` |

## Obsidian 目录结构

配置 `obsidian_vault_path` 后，插件会在保管库下自动创建以下目录：

```
<你的 Obsidian 保管库>/
├── 日记/
│   └── 2026/
│       └── 2026_04/
│           └── 2026-04-29.md
├── 笔记/
│   ├── 游戏笔记/
│   │   └── 所向之未来.md
│   ├── 读书笔记/
│   │   └── xxx.md
│   ├── 影视笔记/
│   ├── 音乐笔记/
│   ├── 追剧笔记/
│   └── 个人健康与运动档案.md
├── 财务/
│   └── 2026-04.md
└── 附件/
    └── img_20260429_a1b2.png
```

- 日记按 `年/月` 分目录，文件名为日期
- 笔记按类型分文件夹，文件名为笔记标题
- 图片自动保存到 `附件/` 目录
- 留空 `obsidian_vault_path` 则不写入 Obsidian
